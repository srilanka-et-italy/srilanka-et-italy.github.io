# Sri Lanka ET Italy - High-End Fusion Restaurant Website

## 🌟 The Silk Road & The Marble Way

A high-end, minimalist website for Sri Lanka ET Italy fusion restaurant located at Schützenverein Wolfegg. This website combines the culinary traditions of Sri Lanka and Italy in a stunning, Apple-inspired design.

## ✨ Features

### 🎨 Design Concept
- **Obsidian Black Background**: Deep, elegant black (#0a0a0a) creating a premium atmosphere
- **Brushed Gold Typography**: Luxurious gold accents (#d4af37) matching the Sri Lanka logo
- **Italian Color Accents**: Subtle red and green touches
- **Clean Typography**: Inter font family with large headlines and generous spacing
- **Apple-Style Aesthetics**: Bento grid layouts with perfect rounded corners

### 🌐 Multilingual Support
Complete translations in 4 languages:
- 🇩🇪 German (Deutsch)
- 🇬🇧 English
- 🇮🇹 Italian (Italiano)
- 🇱🇰 Sinhala (සිංහල)

Language switching is seamless without page reload, preserving user experience.

## 🚀 Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/Srilanka-er-italy/Srilanka-er-italy.github.io.git
cd Srilanka-er-italy.github.io
```

2. Open with a local server (recommended):
```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve

# Using PHP
php -S localhost:8080
```

3. Open your browser to `http://localhost:8080`

### GitHub Pages Deployment

The website is automatically deployed via GitHub Pages:
- **URL**: https://srilanka-er-italy.github.io
- **Branch**: Deploy from `main` or `gh-pages` branch
- **Custom Domain**: Can be configured in repository settings

## 🔧 Customization

### Adding Menu Items
Edit the `index.html` file and add items in the appropriate category section:
```html
<div class="bento-item reveal-up" data-item="new-item">
    <div class="item-image">
        <div class="image-placeholder burger-img"></div>
    </div>
    <div class="item-content">
        <h3 class="item-name">Item Name</h3>
        <p class="item-price">Price €</p>
    </div>
</div>
```

### Adding Translations
Edit `script.js` and add translations in the `translations` object:
```javascript
translations: {
    de: { /* German */ },
    en: { /* English */ },
    it: { /* Italian */ },
    sin: { /* Sinhala */ }
}
```

### Customizing Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --obsidian-black: #your-color;
    --brushed-gold: #your-color;
}
```
